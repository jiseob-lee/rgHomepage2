Privacy Policy

This website (hereinafter referred to as the “Site”) establishes and discloses the following Privacy Policy to protect users' personal information in accordance with the Personal Information Protection Act and to promptly and smoothly handle related grievances.

1. Items of Personal Information Collected

The Site collects the following personal information to provide comment functionality.

Required Items

- Name or Nickname (Purpose of Comments)

- Password (for comment modification/deletion verification)

※ Passwords are stored using one-way encryption (hashing) and the original text is not stored.

* The following personal information items may be automatically generated and collected during the use of internet services.
- IP address, OS information, web browser information

2. Purpose of Personal Information Collection and Use

Collected personal information will not be used for any purpose other than the following:

- Identifying comment authors

- Verifying comment modification and deletion permissions

- Prevention of malicious comments and service misuse/abuse

3. Retention and Use Period of Personal Information

Personal information is retained and used for as long as the comment remains.

If a user deletes a comment, the personal information associated with that comment is immediately deleted.

4. Provision of Personal Information to Third Parties

The site does not provide users' personal information to external parties.

5. Outsourcing of Personal Information Processing

The site does not outsource personal information processing tasks to external parties.

6. Measures to Ensure Personal Information Security

The site implements the following measures in accordance with Article 29 of the Personal Information Protection Act:

- One-way encryption storage for passwords

- Minimization of access permissions

- Server security settings and access control

7. User Rights

Users may terminate personal information processing at any time by editing or deleting comments they have posted.

8. Personal Information Protection Officer

For inquiries regarding personal information processing, please contact the following:

Personal Information Protection Officer: Site Operator

Contact Method: jisblee@gmail.com

9. Changes to the Privacy Policy

This Privacy Policy may be amended due to changes in laws or services. Any changes will be announced on the website.

개인정보 처리방침

본 홈페이지(이하 “사이트”)는 개인정보보호법에 따라 이용자의 개인정보를 보호하고, 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 다음과 같은 개인정보 처리방침을 수립·공개합니다.

1. 수집하는 개인정보 항목

사이트는 댓글 기능 제공을 위해 아래의 개인정보를 수집합니다.

필수항목

- 이름 또는 닉네임 (댓글 용도)

- 비밀번호 (댓글 수정·삭제 확인용)

※ 비밀번호는 단방향 암호화(해시) 처리되어 저장되며, 원문은 저장되지 않습니다.

* 인터넷 서비스 이용과정에서 아래 개인정보 항목이 자동으로 생성되어 수집될 수 있습니다.
- IP주소, OS정보, 웹브라우저정보

2. 개인정보 수집 및 이용 목적

수집한 개인정보는 다음 목적 이외의 용도로는 이용되지 않습니다.

- 댓글 작성자 식별

- 댓글 수정 및 삭제 권한 확인

- 악성 댓글 및 서비스 오·남용 방지

3. 개인정보 보유 및 이용 기간

댓글이 유지되는 동안 개인정보를 보유·이용합니다.

이용자가 댓글을 삭제할 경우, 해당 댓글과 함께 개인정보는 즉시 삭제됩니다.

4. 개인정보의 제3자 제공

사이트는 이용자의 개인정보를 외부에 제공하지 않습니다.

5. 개인정보 처리 위탁

사이트는 개인정보 처리 업무를 외부에 위탁하지 않습니다.

- 비밀번호에 대한 단방향 암호화 저장

- 접근 권한 최소화
